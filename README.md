# 🎮 Sudo-Awake

A 2D top-down action/arcade game built with **GameMaker**, featuring pixel-perfect movement, an in-game cheat terminal, and environmental hazards.

---

## ✨ Features

* **Pixel-Perfect Collisions:** Smooth character movement utilizing loop-based collision checks to completely prevent wall clipping.
* **Cheat Terminal:** An in-game console allowing developers/players to execute commands (e.g., modifying character speed with strict input validation and limits).
* **Health & Damage System:** A dynamic, overhead healthbar that tracks player HP, handles environmental damage (acid pools), and triggers death mechanics at 0 HP.

---

## 🛠️ Tech Stack & Tools

* **Engine:** GameMaker (GML — GameMaker Language)
* **Version Control:** Git + GitHub (optimized with GM-specific `.gitignore`)
